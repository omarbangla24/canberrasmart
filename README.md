# Canberra Smart Energy

Static website for Canberra Smart Energy - your local solar professionals in ACT and NSW.

## Pages

- **Home** - `/`
- **About** - `/about/`
- **Services** - `/services/`
- **Gallery** - `/gallery/`
- **Contact** - `/contact/`

## Deployment

This site is configured for GitHub Pages. Push to the `main` branch and enable GitHub Pages in repository settings.

## Structure

Each page is served as a clean URL using the folder pattern:
- `about/index.html` → `/about/`
- `services/index.html` → `/services/`
- `gallery/index.html` → `/gallery/`
- `contact/index.html` → `/contact/`
